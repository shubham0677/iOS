
Demo iOS app I made following apple's official tutorial to learn key concepts of iOS development.

##The app supports:
- Viewing meals in a table view using custom table cell
- Adding a meal
- Deleting a meal
- Editing a meal
- Save and load meals using NSCoding protocol
